# Hey, I'm Seb.

UBC Mechanical Engineering student on exchange at NTU Singapore. I build things across the full stack: software, hardware, whatever it takes. Most of my projects started as personal problems I got frustrated enough to fix. At some point I realised that if I had the problem, someone else probably did too. So I built it properly, for both of us.

---

**[BackBreeze](https://github.com/seb-krols/back-breeze)** — Singapore is humid and my back was constantly soaked. A parametric TPU mesh generator with gradient density control, airflow analysis, and a live 3D web configurator.

**[Spotlore](https://github.com/seb-krols/spotlore-showcase)** — Started as a hand-coded Valentine's Day gift with 51 locations delivered with flowers. Rebuilt it into a full product: cinematic map tours, six themes, password-protected sharing.

**[GraphConvert](https://github.com/seb-krols/graphconvert)** — AI tools can't read graphs, they just see images. Right-click any graph, get precise (x, y) data points in JSON, CSV, or TXT. Live on the Chrome Web Store.

**[Oh Shit It's Due](https://github.com/seb-krols/oh-shit-its-due)** — Canvas notifications are too polite. Three escalating panic levels, VAPID push that works when the app is closed, Canvas tokens secured server-side.
